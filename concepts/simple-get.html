Simple GET

Load an HTML fragment from the server and swap it into the page — no JavaScript required.

hx-get hx-target hx-swap
Browser
User clicks button
GET /hello.html
Server
Returns HTML fragment
HTML snippet
DOM
Swapped into #target
<!-- The button that triggers the request -->
<button
  hx-get="/hello.html"
  hx-target="#result">
  Load Content
</button>

<!-- Where the response goes -->
<div id="result"></div>

Key Points

1
htmx intercepts the click and fires an XHR GET request automatically
2
The server returns a plain HTML fragment — not JSON, not a full page
3
htmx swaps the response HTML into #result — no page reload
4
Zero JavaScript written by you — just HTML attributes
