Swap Styles

hx-swap controls exactly where in the DOM the server response is placed.

hx-swap
innerHTML (default)
Replaces the contents of the target
Before: <div id="t">Old</div> After: <div id="t">New</div>
outerHTML
Replaces the target element itself
Before: <div id="t">Old</div> After: <section>New</section>
beforeend
Appends after existing content
Item 1 Item 2 + New item
afterbegin
Prepends before existing content
+ New item Item 1 Item 2
beforebegin
Inserts before the target element
+ Inserted here <div id="t">Target</div>
delete
Removes the target from the DOM
Other content <div id="t">Gone</div>
