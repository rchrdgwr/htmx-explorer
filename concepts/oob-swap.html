Out-of-Band Swap

One server response updates multiple, unrelated parts of the page simultaneously.

hx-swap-oob hx-target
Response contains two elements
Server response body
<p>Main content</p>

<div
 id="status"
 hx-swap-oob="true">
 Updated!
</div>
→
Normal swap
into #main-content
→
OOB swap
finds #status anywhere on page
<!-- Normal response target -->
<div id="main-content">...</div>

<!-- Anywhere else on the page -->
<div id="status">idle</div>

<!-- Inside the server response -->
<p>Main content loaded!</p>
<div id="status" hx-swap-oob="true">Updated!</div>

Key Points

1
The main response body goes to the normal hx-target
2
Any extra element with hx-swap-oob="true" is matched by its id and swapped independently
3
No limit on how many OOB elements one response can contain
4
Great for updating a notification badge, status bar, or cart count alongside the main content
