POST Form

Submit a form via htmx — serializes all fields and POSTs without a page reload.

hx-post hx-target hx-swap

Traditional form submit

  • Full page reload
  • Browser navigates away
  • Scroll position lost
  • Must redirect after POST

htmx hx-post

  • No page reload
  • Response swapped inline
  • Scroll position kept
  • Server returns HTML fragment
<!-- Just add hx-post to the form tag -->
<form
  hx-post="/submit"
  hx-target="#result"
  hx-swap="innerHTML">
  <input name="name" type="text">
  <button type="submit">Submit</button>
</form>

<div id="result"></div>

Key Points

1
htmx automatically serializes all form fields into the POST body
2
The server reads the fields and returns an HTML fragment — not JSON
3
hx-post can also go on any button or element, not just the form tag
4
Use htmx-indicator class to show a spinner while the request is in flight
