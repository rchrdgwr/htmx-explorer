Loading Indicators

htmx adds .htmx-request to the triggering element while a request is in flight. Use CSS to show a spinner - no JavaScript needed.

Inline spinner (child of button)

The spinner is inside the button with class="htmx-indicator".
CSS rule .htmx-request .htmx-indicator makes it visible automatically.

External indicator

hx-indicator="#ext-indicator" points to a spinner elsewhere on the page.
htmx adds .htmx-request directly to that element.
