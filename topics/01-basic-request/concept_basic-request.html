1. Basic Request

hx-gethx-targethx-swap

Replace part of a page without writing JavaScript. This is the core idea of htmx — HTML attributes do the work.

Browser
User clicks button
GET /hello
Server
Returns HTML fragment
HTML snippet
DOM
Swapped into #result
<button
  hx-get="/hello"
  hx-target="#result"
  hx-swap="innerHTML">
  Say Hello
</button>

<div id="result"></div>
Key insight: The server returns HTML, not JSON. htmx drops it straight into the DOM — no parsing, no templating on the client.

Key Points

1
htmx intercepts the click and fires an XHR GET request automatically
2
The server returns a plain HTML fragment — not a full page, not JSON
3
htmx swaps the response into #result with no page reload
4
Zero JavaScript written by you — just two HTML attributes
