Targeting

The same response can land in completely different places depending on hx-target.

Target by ID

hx-target="#box-a"
#box-a
Waiting…

Target a different element

hx-target="#box-b" — same response, different destination
#box-b
Waiting…

Target self

hx-target="this" + hx-swap="outerHTML" — the element replaces itself

Target closest ancestor

hx-target="closest tr" + hx-swap="outerHTML" — replaces the whole row
