12. Infinite Scroll / Lazy Loading

hx-trigger="revealed"hx-swap="outerHTML"

Load content on demand as the user scrolls. hx-trigger="revealed" fires when an element scrolls into the viewport.

Scroll
Sentinel enters viewport
GET /items?page=2
Server
Returns next batch + new sentinel
outerHTML swap
DOM
Items appended, sentinel replaced
<!-- Item list -->
<ul id="items">
  <li>Item 1</li>
  <li>Item 2</li>
  <!-- Sentinel: replaces itself with next page + new sentinel -->
  <li hx-get="/items?page=2"
      hx-trigger="revealed"
      hx-swap="outerHTML">
    Loading more...
  </li>
</ul>
The sentinel pattern: The last item in the list is a placeholder that watches for visibility. When it appears, it loads the next page and replaces itself - either with more items + a new sentinel, or nothing if there are no more pages.

Key Points

1
revealed uses IntersectionObserver under the hood - efficient and battery-friendly
2
The sentinel replaces itself with the next page of content via hx-swap="outerHTML"
3
When the last page is reached, the server returns content without a new sentinel - scrolling stops
4
Also great for lazy-loading images, charts, or expensive widgets below the fold
